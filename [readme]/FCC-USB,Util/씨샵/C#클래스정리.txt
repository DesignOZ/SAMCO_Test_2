ConcurrentQueue 클래스
: 멀티쓰레딩 환경에서 큐를 보다 간편하게 사용할 수 있는 클래스이다.

AutoResetEvent 클래스
: 이벤트를 기다리는 스레드에게 신호를 보내 하나의 스레드만 통과시키고 나머지 스레드들은 다음 신호를 기다리게 하는 클래스이다.

 ex> 스레드 A가 WaitOne() 메소드를 써서 대기하고 있다가, 다른 스레드 B에서 Set() 메소드를 호출하면, 스레드 A는 대기 상태를 해제하고 다음 문장을 실행 할 수 있게 된다.

Out 키워드
: 함수외부에서 인자값을 초기화 하지않고 쓰고 싶을 때 사용하는 키워드
ex> 
int x;
CallByRef(out x);
--> x값이 아무것도 없이 초기화가 안된 상태에서 그냥 인자로 쓸 수 있다.
